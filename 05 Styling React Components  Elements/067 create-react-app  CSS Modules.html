When using create-react-app v2+ (you can check that by running create-react-app -V), support for CSS modules is already built-in!

You don't need to eject then (though you still can - that won't be a problem). Instead, you can use CSS modules as described here: https://github.com/facebook/create-react-app/blob/master/packages/react-scripts/template/README.md#adding-a-css-modules-stylesheet
